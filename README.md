# platformer_proto

Прототип 3D-платформера на Rust с использованием `macroquad`.

## Требования

- Установленный Rust (через [rustup](https://rustup.rs/))
- `cargo` в `PATH`

Проверка установки:

```bash
rustc --version
cargo --version
```

## Запуск проекта

1. Клонируй репозиторий и перейди в папку проекта:

```bash
git clone <repo-url>
cd <repo-folder>
```

2. Запусти проект в debug-режиме:

```bash
cargo run
```

3. Для более высокой производительности запусти release-сборку:

```bash
cargo run --release
```

## Полезные команды

- Проверка компиляции без запуска:

```bash
cargo check
```

- Сборка бинарника:

```bash
cargo build
```

- Сборка оптимизированного бинарника:

```bash
cargo build --release
```

## Управление в игре

- `Tab` — открыть/закрыть меню
- `W/A/S/D` — перемещение
- `Space` — прыжок
- `Left Shift` — ускорение
- `G` — включить/выключить сетку
- `X` / `Y` — инверсия осей камеры
- `F3` — отладочная информация
